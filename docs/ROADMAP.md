# Roadmap

## MVP-1: Safe Local CRM Foundation

- Project structure and public repository hygiene.
- Prisma SQLite schema.
- AI settings page and onboarding wizard.
- Resume text analysis.
- First search profile from resume analysis.

## MVP-2: Vacancy Intake and Analysis

- Manual vacancy entry.
- Dual AI routing: analysis, writer, reviewer, fast, premium.
- AI vacancy fit analysis.
- Cover letter generation.
- Vacancy statuses, next actions, testing fields, and follow-up draft.

## MVP-3: Usable HH Search Workflow

- PDF resume text extraction without OCR.
- Resume detail page with manual editing.
- Confirmed facts for safe cover letter generation.
- Browser-based hh search through Playwright persistent context.
- Search runs and search history.
- Deduplication by hh link or vacancy id.
- Sequential AI analysis of collected vacancies.
- Recommended vacancies screen for manual applications.

## Next

- Employer dossier.
- Archive history and reasons.
- Repeated publication tracking.
- Company checks through public sources.
- Extended follow-up assistant.
- Interview stages and notes.
- Safer profile update flow from edited resume.
- OCR option for scanned PDF resumes.
- Better region selection for hh area ids.
- Export and backup tools.
